## Section 3: Safety & Consumable Replenishment

### 1. Safety Hazard â€“ Methylated Spirit

The Grade 7 syllabus (7.43) lists **â€œMethylated spiritâ€** for the photosynthesis starch test. This is a **high-risk flammable chemical** and **should not be included in a student kit**.

The kit should only provide **iodine solution, beakers, and test tubes**. Schools must be instructed that:

- Methylated spirit is **for teacher demonstration only**.  
- It must be used with **extreme caution**, in a **well-ventilated area**, away from open flames.  
- Appropriate **fire safety equipment** must be available.
